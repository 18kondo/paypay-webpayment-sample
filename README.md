# PayPay Webpayment Sample

## Installation

```sh
$ npm install
```

## env

```sh
$ export API_KEY="REPLACE_WITH_YOUR_API_KEY"
$ export API_SECRET="REPLACE_WITH_YOUR_SECRET_KEY"
$ export MERCHANT_ID="REPLACE_WITH_YOUR_MERCHANT_ID"
```


## Running the app

```sh
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
